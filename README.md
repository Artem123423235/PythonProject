# учебный проект Python
## Цель проекта
Этот проект предназначен для обработки данных о транзакциях, позволяя фильтровать и сортировать их по состоянию и дате.

## Установка

Для установки и использования проекта, выполните следующие шаги:

1. Клонируйте репозиторий:
git clone https://github.com/your_username/my_project.git


2. Перейдите в директорию проекта:
cd my_project


3. Убедитесь, что у вас установлены все необходимые зависимости. (Уточните их, если они есть.)

## Использование

### Функции

1. `filter_by_state(data, state='EXECUTED')`
- **data**: Список словарей с данными о транзакциях.
- **state**: (опционально) Значение для фильтрации по ключу 'state'. По умолчанию 'EXECUTED'.
- **Возвращает**: Новый список словарей с указанным состоянием.

**Пример использования**:
```python
data = [
    {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
    {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'},
    {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}
]
executed_transactions = filter_by_state(data)

sort_by_date(data, descending=True)

data: Список словарей с данными о транзакциях.
descending: (опционально) Порядок сортировки. По умолчанию True (по убыванию).
Возвращает: Новый отсортированный список.
Пример использования:

sorted_transactions = sort_by_date(data)
